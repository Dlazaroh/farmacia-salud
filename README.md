📌 Farmacia Salud

📝 Descripción
Esta farmacia virtual resuelve la dificultad de acceder a medicamentos y artículos variados a cualquier hora, permitiendo comprarlos de forma rápida y sin recargar la página. Sus características principales son un carrito interactivo con vista detallada de los ítems agregados, modales informativos y un selector de modo visual (claro, oscuro y sepia), aportando accesibilidad personalizada y una experiencia de usuario fluida estructurada con HTML5, SCSS y JavaScript.
